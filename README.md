# Task Manager App

## Description
A simple task manager application

## Features
- ADD tasks
- Mark tasks status
- Delete Tasks
- User Friendly

## How to Run
1. Clone this repository: git clone git@github.com:MRJoe9999/task-manager.git
2. Navigate to the folder: Homework
3. run npm install 
4. run the application using npm start command
